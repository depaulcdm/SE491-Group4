# SE491-Group4 B.E.S.S.T. (Bilingual Educational Speech Support Tool)

## **Abstract**
B.E.S.S.T. (Bilingual Educational Speech Support Tool) is a web application designed to automate the manual, time taking process Speech Language Pathologists face when prepping therapy materials. The application enables them to instantly generate custom worksheets that fits their specific criteria.

## **Persona**
Our Speech Therapy Application aims to streamline the workflow of a Speech Language Pathologists. Our primary persona, Emily, is stuck with the tedious task of having to create worksheets manually for her students, she tries her best to implement the proper phonetic and syllabic needs of her students but is very time consuming. Our application solves this matter by allowing the user to search and filter words based on length, sound, and syllable structure. The user will be able to generate and export these unique worksheets based on their unique input. This will speed up the processes of prepping worksheets for student and allows the Speech Language Pathologist to deliver more learning towards students. 

## **Tech Stack**
**Front End:** Built with React for interactive user experience and fast worksheet generation.

**Backend:** .NET to handle complex filtering and logic

**DataBase:** Utilizing the CMU Pronouncing Dictionary for English and a custom dictionary for Spanish.
